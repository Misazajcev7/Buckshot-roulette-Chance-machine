# Buckshot Roulette Chance Machine

A web-based probability counter and voice-controlled assistant designed for the game *Buckshot Roulette*. Track rounds, calculate exact risks, and use hands-free voice commands to stay ahead of the Dealer.

## Features

- **Dynamic Probability Calculation:** Real-time percentage breakdowns for Blank (Blue) and Live (Red) shells.
- **Voice Assistant:** Includes an interactive assistant ("Sam" for English, "Rick" for Russian) predicting the next outcome.
- **Interactive Bullet Icons:** Click directly on the bullet visual indicators to quickly decrease the remaining round count.
- **Lock/Reset Safeguards:** Lock your initial load ratio or reset the board instantly.
- **Multilingual Voice Controls:** Supports both English and Russian voice navigation.

## Voice Commands Reference

### English (EN)
- `lock` / `freeze`: Lock the current configuration.
- `reset` / `clear`: Clear all entries and start over.
- `live` / `add live`: Increase live round count.
- `blank` / `add blank`: Increase blank round count.
